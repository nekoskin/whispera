Настройки автообновления

Автоматически обновлять Geo базы данных через заданный интервал
Geo базы данных:
  • GeoIP: База данных для определения страны по IP адресу
  • GeoSite: База данных для определения страны по домену
  • Базы обновляются автоматически из репозитория v2fly
  • Используются для правил маршрутизации по стране

Логи сервера


                    

                

            

        

    


    
    

        

            

                
 Создать нового пользователя

                
            

            

                
                

                    
 Основная информация

                    

                        
                        
                        
                            
                            Только латинские буквы, цифры и подчеркивание. Пример: user1, client_123
                        
                    

                    

                        
                        
                        Используется для уведомлений и восстановления доступа
                    

                


                
                

                    
 Настройки обфускации

                    

                        
                        
                        
                            QUIC: Быстрый и эффективный, рекомендуется для большинства случаев

                            HTTP/2: Стабильный и надежный, хорош для проблемных сетей

                            WebSocket: Универсальный, работает через большинство прокси и файрволов
                        
                    

                    

                        
                        
                        
                            Browser: Универсальный профиль, подходит для всех случаев

                            VK/Yandex/Mail.ru: Специализированные профили для обхода блокировок в России
                        
                    

                    

                        
                        
                            
                            Система автоматически выберет оптимальный профиль для каждого подключения на основе сетевых условий и типа блокировки
                        
                    

                


                
                

                    
 Ограничения

                    

                        
                        
                        
                            
                            0 = безлимит. Укажите число в GB (например: 10, 50, 100). 
                            После достижения лимита доступ будет заблокирован.
                        
                    

                    

                        
                        
                        
                            Оставьте пустым для бессрочного доступа. После указанной даты пользователь не сможет подключиться.
